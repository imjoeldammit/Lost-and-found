How This System Works
🔌
Hardware Layer
A Raspberry Pi Pico W runs MicroPython. When the trigger fires, the OV2640 captures a JPEG via SPI FIFO. LEDs give instant feedback — red for urgent items, green for normal ones.
☁️
Real AI Identification
The JPEG is sent to Claude Vision API which genuinely analyses the image and returns the object name, category, priority, condition, and a detailed description — not a lookup table.
📋
Campus Registry
Every item gets a unique LF-XXXX ID with timestamp and location. Staff can filter by priority and mark items as claimed. Critical items (laptop, ID, wallet) trigger urgent alerts.
