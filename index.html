How This System Works
🔌
Hardware Layer
Raspberry Pi Pico W + OV2640 camera captures JPEG frames. A trigger button fires the scan. LEDs give instant feedback — red for urgent, green for normal.
🧠
AI Vision
Real webcam image is sent to AI Vision API which genuinely identifies the object, assigns priority, color, condition, and writes a handling description for staff.
☁️
Cloud Registry
Items are saved to Firebase Firestore in real time. Registry persists across sessions. Students can search and claim items. Staff get email alerts for critical items.
